# BrightTV_Analytics_Copilot

This project aims to build an Analytics Copilot that answers business questions using the BrightTV case study data. The copilot uses Databricks SQL to analyze subscriber demographics and viewership data, generate insights, and support data-driven decision-making.

The project demonstrates SQL querying, data cleaning, data analysis, and business intelligence techniques. Future enhancements will include Python and AI capabilities to enable users to ask questions in natural language and receive automated insights.
The BrightTV Analytics Copilot was developed using Databricks Genie to provide answers to business questions related to subscriber demographics and viewing behaviour.

The copilot uses BrightTV subscriber and viewership datasets to generate insights around customer profiles, engagement trends, and content consumption patterns.

Key capabilities include:

* Subscriber demographic analysis
* Viewership trend analysis
* Channel performance analysis
* Customer segmentation insights
* Data quality observations
